##  Project Overview

**Local Dry Fruits** is a full-stack e-commerce web application designed to connect **customers** with **local dry fruit shopkeepers**.
Customers can browse, reserve, and order dry fruits online, while shopkeepers can manage products, stock, and orders through a dedicated dashboard.

This project focuses on **supporting local businesses** while providing a smooth and modern online shopping experience.

---

##  Key Features

###  Customer Module

* User registration & login
* Browse dry fruits with images and prices
* Add products to cart
* Place orders & checkout
* View order history
* Profile management

---

### Shopkeeper Module

* Secure shopkeeper login
* Add, edit, and delete products
* Update available stock
* Freeze & revert stock during orders
* View customer orders
* Update order status

##How to Run Locally

1. Install **XAMPP**
2. Clone this repository:

   ```bash
   git clone https://github.com/Siddartha-balla/dryfruit-ecommerce-site.git
   ```
3. Move the project folder to:

   ```
   C:\xampp\htdocs\
   ```
4. Start **Apache** and **MySQL**
5. Import the database into **phpMyAdmin**
6. Open in browser:

   ```
   http://localhost/dryfruitReserve/
   ```
